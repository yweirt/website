Open datasets created with 3dfier

A few Dutch cities that we reconstructed with 3dfier by combining the BGT dataset (large-scale topographic dataset of the Netherlands) (license is CC BY 4.0) and the AHN3 datasets (laser altimetry) (license is CC0 1.0).

The dataset below are all available under the Creative Commons CC0 Public Domain Dedication.

We offer different formats:

  • OBJ: simple format useful for visualisation, use this to visualise the models. The free software MeshLab works great, and is cross-platform.
  • CityGML: the file will be much bigger than the OBJ. You can visualise it with our free software Azul (MacOS) or FZKViewer (Windows).
  • IMGeo: Dutch format, more details there. There are no proper visualiser for this at this moment.

If you want to use 3dfier yourself, you can follow that handy tutorial.
